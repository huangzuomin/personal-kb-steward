# 📑 [The New York Times adopts AI tools in the newsroom](https://www.theverge.com/news/613989/new-york-times-internal-ai-tools-echo)



- **核心要点：**
  - 《纽约时报》已批准新闻编辑室使用AI工具，涵盖编辑、总结、编程及写作辅助等多个方面。
  - 引入内部AI工具 **Echo**，用于总结文章、简报等内部信息。
  - 发布新的编辑指南，鼓励使用AI进行编辑建议、生成摘要、社交媒体推广文案和SEO标题。
  - AI工具也可用于生成新闻测验、引用卡片、常见问题解答，甚至为采访提供问题建议。

- **使用限制：**
  - AI**不应**用于起草或大幅修改文章。
  - **禁止**使用AI规避付费墙或输入第三方受版权保护的材料。
  - **禁止**发布未经明确标注的AI生成图像或视频。

- **官方立场：**
  - 重申新闻采写和编辑工作始终由人工记者负责。
  - AI辅助工作的产出必须基于记者核实的事实，并由编辑审阅。

- **其他获批工具：**
  - GitHub Copilot (编程)
  - Google Vertex AI (产品开发)
  - NotebookLM
  - NYT的ChatExplorer
  - OpenAI非ChatGPT API
  - 部分亚马逊AI产品

- **背景信息：**
  - 《纽约时报》目前正起诉OpenAI和Microsoft，指控其在未经许可的情况下使用时报内容训练ChatGPT。
  - 其他新闻机构也在不同程度地引入AI工具。
