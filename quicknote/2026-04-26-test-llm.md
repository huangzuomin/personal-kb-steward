# 测试 LLM 标题生成

测试目的：验证 DeepSeek 模型在 mindseed-grow 场景下能否生成有意义的 seed 标题。

## 测试内容

- 来源：quicknote/2026-04-26-test-llm.md
- 主题：AI 搜索工具对新闻业的影响
- 标签：#seed

## 关注点

1. 标题是否从原始内容提取，而不是词频拼接
2. summary 是否准确归纳了核心议题
3. growth_directions 是否有具体方向

## 预期

如果 LLM 正常工作，seed 标题应该类似"AI 搜索工具如何改变新闻发现方式"，而不是"动态议题：AI / 搜索 / 新闻"。
